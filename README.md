# University Portal Project

Αυτή είναι μια διαδικτυακή εφαρμογή που αναπτύχθηκε στα πλαίσια του μαθήματος "Web & Mobile Applications Development" (CN5006). Πρόκειται για μια πύλη διαχείρισης πανεπιστημίου που υποστηρίζει δύο ρόλους χρηστών: Φοιτητές και Καθηγητές.

## Δυνατότητες (Features)
* **Δημόσια Πύλη:** Πληροφορίες για το campus και ενσωματωμένος διαδραστικός χάρτης (Leaflet.js).
* **Ταυτοποίηση:** Σύστημα Εγγραφής/Σύνδεσης (Sign up/Login) με επαλήθευση ρόλου.
* **Έλεγχος Πρόσβασης Βάσει Ρόλου (RBAC):**
    * **Φοιτητές:** Μπορούν να βλέπουν τα μαθήματα, να υποβάλλουν εργασίες και να βλέπουν τις βαθμολογίες τους.
    * **Καθηγητές:** Μπορούν να δημιουργούν μαθήματα, να αναρτούν εργασίες και να βαθμολογούν τις υποβολές.
* **Βάση Δεδομένων:** Πλήρως σχεσιακή βάση MySQL με χρήση Foreign Keys.

## Τεχνολογίες που Χρησιμοποιήθηκαν
* PHP (Native)
* MySQL / MariaDB
* HTML5 / CSS3 / JavaScript
* Leaflet.js (Χάρτης)
* FontAwesome (Εικονίδια)
